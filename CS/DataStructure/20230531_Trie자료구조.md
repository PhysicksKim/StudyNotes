# Trie 에 대해서  
  
### Trie 란?  
문자열을 저장하고 효율적으로 탐색하기 위한 트리 형태의 자료구조  
re**trie**val(검색) tree 에서 유래한 이름이다.  
   
### 유용한 예시  
> "ca" 로 시작하는 단어는?  
  
그러면 어떻게 "ca"로 시작하는 단어를 찾을 수 있을까?   
만약 Hash를 사용한다면 뭔가 머리가 복잡해진다. 이걸 어떻게 Hash로 구현하지?    
   
일일이 "c" hash 만들고 "ca" 해시 만들고 하는 식이면    
"catabolism" 같은 긴 단어면 어쩌려고?    
    
그런데 Trie를 사용하면    
그냥 root부터 c->a 로 타고 들어간 다음     
그 하위 모든 단어를 완성시켜서 반환해버리면 된다    
      
### 구조는 단순하다  
구조는 그냥 그림만 봐도 Tree 구조만 구현할 줄 알면 단순히 만들어 낼 수 있다  
구현 코드나 구현 노하우는 생략한다.
